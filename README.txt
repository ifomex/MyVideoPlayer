O Aplikaci
----------
Aplikace MyVideoPlayer je jednoduchý video přehrávač pro platformu Android. Umožňuje přehrávat základní formáty videí (mp4, 3gp,...). Aplikace také umožňuje vytvářet libovolné playlisty.
